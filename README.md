## 特徴

- 本物の付箋紙のように自由に配置できます
- ブラウザ(IndexedDB)にデータを保存します。

    バックアップや他のPCへのデータ移行が必要な場合は、右上のメニューからエクスポートしたファイルを移行先にインポートしてください。
